# New mGBA
New mGBA 基于 borealis UI 和 mGBA 模拟器核心开发，旨在为用户提供一个现代化、易用且功能丰富的游戏模拟器界面。

# 开发计划
## 主界面选项
1. 游戏
2. 设置
3. 关于
4. 退出

## 选项子菜单
### 游戏
- [] 选择游戏文件 -> 点击后打开文件列表界面，
- [] 收藏夹 -》 打开收藏夹列表界面
- [] 最近游玩游戏 -》 打开最近游玩游戏列表界面
以上界面都包含游戏图标、名称、最近游玩时间等信息，按A进入游戏界面，按Y打开游戏设置界面
以上界面可以直接复用UI， 并且支持在列表视图、网格视图、水平视图之间切换

#### 游戏设置界面
1. 打开游戏（如果游戏打开后进入的游戏设置界面，则显示为返回游戏）
2. 金手指 -》 打开金手指设置界面（先打开空页面，后续根据需要添加设置项）
3. 设置（就是下面的设置选项）
4. 返回选单（如果游戏打开后进入的游戏设置界面，则显示为关闭游戏）
